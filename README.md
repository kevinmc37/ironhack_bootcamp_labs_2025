### 📂 Índice de Laboratorios y Progresión Técnica

Los proyectos están organizados en subcarpetas locales dentro de este repositorio. Puedes navegar por ellos cronológicamente según la competencia adquirida:

#### 1. Fundamentos de Java y Programación Orientada a Objetos (POO)
*   **[Core Java Basics](./lab-java-basics-es):** Control de flujos, manipulación de Strings, arrays, depuración de errores y pruebas unitarias iniciales con **JUnit**.
*   **[Interfaces & Abstract Classes](./lab-java-interfaces-and-abstract-classes-es):** Implementación de modelado del mundo real mediante herencia, clases abstractas y polimorfismo. Diseño estructurado para evitar la duplicidad de código.

#### 2. Diseño de Bases de Datos y Persistencia
*   **[Database Normalization & Advanced SQL](./lab-java-normalization-ddl-aggregation-es):** Diseño y optimización de esquemas relacionales. Normalización de tablas, operaciones DDL, consultas complejas empleando uniones (`JOIN`) y funciones de agregación.
*   **[Intro to JPA & Spring Data](./lab-java-intro-to-jpa-with-spring-data-es):** Primer contacto con el mapeo objeto-relacional (ORM). Configuración de entidades con Hibernate y persistencia automática mediante interfaces `JpaRepository`.
*   **[JPA Relationships & Inheritance](./lab-java-jpa-relationships-and-inheritance-es):** Modelado avanzado de bases de datos mediante relaciones complejas (`@OneToMany`, `@ManyToOne`, `@ManyToMany`) y estrategias avanzadas de herencia de entidades (`JOINED`, `SINGLE_TABLE`).

#### 3. Ecosistema Spring Boot y APIs REST
*   **[Spring Boot Fundamentals](./lab-java-springboot-fundamentals-es):** Configuración e inicialización de microservicios. Arquitectura clásica en capas (`Controller`, `Service`, `Repository`) y mecanismos de inyección de dependencias.
*   **[REST API Development](./lab-java-springboot-rest-api-es):** Construcción y exposición de endpoints HTTP estándar, validación de payloads de entrada, códigos de estado HTTP semánticos y estructuración de respuestas JSON.

Welcome to my centralized repository containing all the laboratories, projects, and code challenges completed during the Full-Stack Web Development Bootcamp at Ironhack. This space serves as a transparent log of my technical evolution, scaling from bare-metal Object-Oriented Programming (OOP) in native Java, moving up to relational database design and the Spring Boot framework, and culminating in full-tier enterprise backend architectures.

### 📂 Repository Index & Technical Progression

Every assignment is encapsulated in its own subfolder inside this root workspace. You can explore them step-by-step:

#### 1. Java Core & Object-Oriented Programming (OOP)
*   **[Core Java Basics](./lab-java-basics-es):** Control flow mechanisms, String data handling, arrays, debugging techniques, and programmatic isolation tests using **JUnit**.
*   **[Interfaces & Abstract Classes](./lab-java-interfaces-and-abstract-classes-es):** Real-world structural abstraction modeling through class inheritance, abstraction layers, and polymorphism to build robust, DRY (Don't Repeat Yourself) codebases.

#### 2. Data Persistence & Relational Design
*   **[Database Normalization & Advanced SQL](./lab-java-normalization-ddl-aggregation-es):** Relational schema architecture and performance tuning. Structural table normalization, Data Definition Language (DDL) scripting, complex query pipelines utilizing multi-table `JOIN` statements and aggregation functions.
*   **[Intro to JPA & Spring Data](./lab-java-intro-to-jpa-with-spring-data-es):** Foundations of Object-Relational Mapping (ORM). Creating Hibernate entity blueprints and structuring repository patterns via `JpaRepository` interfaces.
*   **[JPA Relationships & Inheritance](./lab-java-jpa-relationships-and-inheritance-es):** Mapping interconnected data configurations via rich relational annotations (`@OneToMany`, `@ManyToOne`, `@ManyToMany`) alongside advanced ORM inheritance mappings (`JOINED`, `SINGLE_TABLE`).

#### 3. Spring Boot & RESTful Architectures
*   **[Spring Boot Fundamentals](./lab-java-springboot-fundamentals-es):** Bootstrapping server environments. Layered system architecture setup (`Controller`, `Service`, `Repository`) and dependency injection control flows.
*   **[REST API Development](./lab-java-springboot-rest-api-es):** Exposing robust HTTP service channels, processing input data validation tokens, implementing semantic HTTP response status blocks, and shaping clean JSON response layouts.
